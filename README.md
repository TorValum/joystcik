Наземная станция управления БПЛА
Проект включает:
   - Наземную станцию на Python (pygame, pyserial)
   - Прошивку для микроконтроллера STM (C, HAL)
   - Графический интерфейс с визуаализацией стиков и команд
   - Протокол обмена данными по UART
     

   1. Наземная станция (Python): Использует pygame для джойстика и pyserial.
   * Протокол: Строго текстовый формат f"{cmd} {arg}\n".
      * Частота отправки: 50 Гц.
      * Диапазоны: Стики масштабируются в значения 500–1500 (согласно текущим настройкам таймеров).
      * Логика: Кнопки (Парашют) имеют приоритет над стиками.
   2. Бортовой компьютер (STM32F4 / C / HAL):
   * UART3 (57600 baud): Обработка в while(1) через HAL_UART_Receive (timeout 0) с накоплением в rx_buffer.
      * Парсинг: Реализован через sscanf(rx_buffer, " %c %d", &cmd, &arg). Пробел перед %c обязателен для очистки буфера от \n.
      * Таймеры (ШИМ):
      * TIM4 (CH1, CH2): Элевоны.
         * TIM3 (CH4): Парашют (пин PC9).
         * Настройки: Prescaler = 107, Period = 2000. (Рабочий диапазон ШИМ: 500–1500).
